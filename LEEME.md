# Mi Álbum · Mundial 2026 (interactivo)

Álbum interactivo generado a partir de tus PDFs. **1111 figuritas.**

## Cómo usarlo

- **Abrí `index.html`** (doble clic). Funciona sin internet, sin servidor.
- Las figuritas aparecen en **gris**. Tocá una para marcar que **la tenés** → aparece la figurita real a color.
- Cada toque extra **suma una repe**. El botón **−** (arriba a la izq.) saca una.
- Pasá el mouse sobre una que tenés para ver **📷** (poné tu propia foto) y **−**.
- **Doble clic** abre la vista grande con controles.
- **Importar fotos en lote**: si el archivo se llama como el código (ej. `ARG1.jpg`, `GER17.jpg`) se pega solo.

## Estructura (filtro de categorías arriba del álbum)

- **⚽ Selecciones** — cada equipo tiene **18 figuritas**:
  - 1–16: jugadores
  - 17: escudo · 18: grupal "We Are"
- **🏆 FWC Clásicos** — campeones históricos (Oro + Plata).
- **🥤 Coca-Cola** — edición Coca-Cola + Pines.
- **✨ Extras** — Extra Stickers + Amarillas (cracks).
- **🗂️ Otros** — escudos/grupales **sin asignar** (de equipos que quedaron por confirmar). Arrastralos/renombralos cuando confirmes el equipo.

## Pestañas

- **Álbum** — por categoría. La tira de banderas/íconos te lleva a cada sección.
- **Faltan** — solo las que te faltan, por sección.
- **Repes** — tus repetidas, listas para cambiar.
- **Resumen** — progreso total, barras por categoría/equipo, y **Exportar / Importar / Reiniciar**.

Tu progreso se guarda en el navegador. Usá **Exportar progreso** (Resumen) para una copia de seguridad.

## De dónde salió cada cosa

- **Jugadores** → `Archivo Definitivo A4 HD…pdf`
- **Escudos** → `Escudos Calidad Oro/Plata V2.pdf`
- **Grupales** → `GRUPALES We Are V2.pdf` / `…Dorado V2.pdf`
- **FWC Clásicos** → `FWC Equipos Clasicos Oro/Plata v4.pdf`
- **Coca-Cola** → `COCACOLA_A4 v3.pdf` + `Pines Coca-Cola.png.pdf`
- **Extras** → `ExtraStickers 4 Versiones…` + `ExtraSticker Amarillas…` (sólo frentes; descarté dorsos y las versiones de color repetidas del mismo jugador)
- **Banderas** → recortadas de cada figurita.
